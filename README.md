# Video Editing App UI 

## Overview

The purpose of this challenge is to build an interactive user dashboard. This dashboard provides functionalities for users to view their basic profile, manage recently created videos, and administer collaborators. The application should be developed using React, Next.js, and Material UI, keeping in line with best coding practices.

### Key Features

#### User Profile Overview:
 - A dashboard page has been designed that prominently displays the user's name and email.
 - A profile picture of the user is seamlessly integrated into the dashboard.
 - The system fetches user profile information from JSON data.
#### Recent Videos Section:
 - This feature displays a list of the user's recently uploaded videos, highlighting the thumbnail, title, and the creation date for each video.
 - Clicking on a video thumbnail opens a drawer in which details information is displayed.
#### Collaborators Management:
 - A collaborators list of avatar with detail view on clicking.
 - The section showcases a comprehensive list of collaborators, detailing their names and designated roles.
 - Users are given a modal to add new collaborators.

## Getting Started

To get the project up and running, follow these steps:

1. Clone the repository: `git clone https://github.com/AibakButt/VideoEditorApp-UI`
2. Navigate to the project directory: `cd user-dashboard`
3. Install the required dependencies: `npm install`
4. Run the application: `npm run dev`

Open your browser and visit `http://localhost:3000` to see the application in action.
